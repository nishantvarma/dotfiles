dotfiles for a minimal linux system

- distro: arch linux
- wm: jwm
- panel: tint2
- launcher: rofi
- terminal: kitty, termite
- shell: zsh
- file manager: vifm, thunar, worker
- editor: sublime, vim
- pdf: atril, llpp, zathura
- audio: audacious, vlc
- video: mpv
- irc: thelounge
- browser: google-chrome, qutebrowser
