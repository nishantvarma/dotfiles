dotfiles for a minimal linux system

- distro: arch linux
- wm: rio, 2bwm
- panel: bmpanel
- launcher: dmenu
- terminal: urxvt
- shell: bash, rc, zsh
- file manager: vifm, thunar
- editor: acme, sublime, (neo)vim
- pdf: llpp
- audio: audacious, vlc
- video: mpv
- irc: hexchat
- browser: firefix
