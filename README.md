dotfiles for a minimal linux system

- distro: archlinux
- wm: 2bwm, rio
- panel: bmpanel
- launcher: dmenu
- terminal: urxvt, screen
- shell: bash, rc
- file manager: vifm, thunar
- editor: acme, sublime, [neo]vim
- pdf: llpp
- audio: audacious, vlc
- video: mpv
- irc: hexchat
- browser: firefox
