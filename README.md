Dotfiles for a minimalistic Linux system.

- Distro: archlinux
- Wm: 2bwm, rio
- Panel: bmpanel
- Launcher: dmenu
- Terminal: tmux, urxvt
- Shell: bash, rc
- File Manager: vifm, thunar
- Editor: acme, sublime, [neo]vim
- Pdf: llpp
- Audio: audacious, vlc
- Video: mpv
- Irc: hexchat
- Browser: firefox
